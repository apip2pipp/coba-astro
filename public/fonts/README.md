# Folder Fonts

Letakkan file font Anda di sini.

## Contoh:
- Poppins-Regular.woff2
- Poppins-Bold.woff2
- Poppins-SemiBold.woff2

## Download Font Gratis:
- Google Fonts: https://fonts.google.com/
- Font Squirrel: https://www.fontsquirrel.com/
- DaFont: https://www.dafont.com/

## Format Yang Direkomendasikan:
- `.woff2` (paling ringan dan modern)
- `.woff` (fallback untuk browser lama)

## Cara Download dari Google Fonts:
1. Buka https://fonts.google.com/specimen/Poppins
2. Klik "Download family"
3. Extract file ZIP
4. Copy file .ttf ke folder ini
5. Atau gunakan converter online untuk convert ke .woff2
